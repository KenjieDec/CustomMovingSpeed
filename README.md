## Features

Customize your **Walk**, **Spring**, **Swim** & **Raft** speed.

<br>
<br>

## Commands

### Show Speed

*Display how fast you are going.*

```shell
showSpeed
```

<br>

### Walk Speed

*Set the walk speed multiplier.*

```shell
# Default Speed 3
setWalkSpeed <Amount>
walkSpeed <Amount>
```

<br>

### Sprint Speed

*Set the sprint speed multiplier.*

```shell
# Default Speed 6
setSprintSpeed <Amount>
sprintSpeed <Amount>
```

<br>

### Swim Speed

*Set the swim speed multiplier.*

```shell
# Default Speed 2
setSwimSpeed <Amount>
swimSpeed <Amount>
```

<br>

### Raft Speed

*Set the speed multiplier of your raft.*

```shell
# Default Speed 1.5
setRaftSpeed <Amount>
```

<br>

### Anchor Raft

*Stop your raft from moving.*

```shell
anchorRaft
```
